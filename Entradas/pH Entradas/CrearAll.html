CrearAll

¿Qué hace?

Versión extendida de Crear, diseñada para aplicar animaciones de entrada controladas a múltiples Mobjects o VGroups. Usa internamente la animación Create (o Crear) sobre cada elemento individual.

Tiene un control inteligente sobre la reproducción de sonidos:

  • Si lag = 0.0: solo se permite un sonido de texto y uno de no texto a la vez.
  • Si lag > 0.1: se permiten varios sonidos simultáneos.

Además, si divide = True, los VGroups se descomponen automáticamente y se aplica Crear a cada uno de sus componentes.

Parámetros

  • *vgrp (Mobjects/VGroups): Lista de objetos a introducir en la escena.
  • **kwargs: Hereda todos los argumentos de Crear, como debug, lag, sound, seed, run_time, etc.
