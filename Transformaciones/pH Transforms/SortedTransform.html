SortedTransform

¿Qué hace?

Divide cada objeto o grupo de objetos en sus elementos más pequeños. Luego transforma el primero del primer grupo con el primero del segundo grupo, el segundo con el segundo, y así sucesivamente.

Si el primer grupo tiene menos elementos, se crean copias de los elementos de este para luego continuar con la asignación de las transformaciones.

Si el segundo grupo tiene menos elementos, el ciclo se repite desde el principio con ese grupo. Por ejemplo, si vgrp1 tiene 6 elementos y vgrp2 solo 2, entonces el 3º de vgrp1 se transforma con el 1º de vgrp2, el 4º con el 2º, el 5º con el 1º nuevamente, etc.

Parámetros

  • vgrp1 (Mobject): Mobject o VGroup a transformar.
  • vgrp2 (Mobject): Mobject o VGroup objetivo de la transformación.
  • **kwargs: Argumentos adicionales de default_args (debug, lag (por defecto 0.0), sound, seed, divide, rand (por defecto False), option, nos).
